# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.53] - 2026-01-15

### Changed
- Updated application version to 1.0.53 (Maintenance release)

## [1.0.52] - 2026-01-13

### Added
- **V2 User Interface**: Completely redesigned, responsive interface with modern dashboard, sidebar navigation, and mobile-first design
- **Project Status Tracking**: Track projects through workflow stages (Draft, Planning, Approved, Shooting, Completed, Archived)
- **Device Library**: Browse and manage equipment database with filtering and search
- **Contacts Management**: Crew roster with roles, contact details, profile photos, and vCard import
- **Owned Gear Tracking**: Personal equipment inventory with quantities and sourcing notes
- **Automatic Gear Rules**: Scenario-triggered equipment additions or removals
- **Enhanced Dark Mode**: Refined dark theme with improved contrast and readability
- **Pink Mode**: Fun accent theme with animated "icon rain" effect
- **Firebase Studio Integration**: Optional cloud sync for real-time collaboration

### Changed
- **Expanded Lens Database**: Added Sirui Anamorphics, NiSi Athena, Kinefinity Mavo, Spirit Lab Pure, and more
- **Redesigned Print Preview**: Sidebar-based print experience with real-time previews
- **Enhanced Command Palette**: `Cmd+K` / `Ctrl+K` for quick actions
- **Interactive Contact Photos**: Drag-and-drop avatar updates with cropping tools

### Fixed
- Project name consistency across documentation (cine-power-planner)
- Added project badges to README for better visibility

## [1.0.0] - Initial Release

### Added
- Equipment power knowledge base with bundled draw figures
- Battery runtime planning with capacity and voltage modeling
- Custom rig configurations for V-Mount, B-Mount, and Gold-Mount
- Offline-first operation with all assets bundled
- Local-only data storage with layered safety nets
- Manual saves, auto-saves, and auto-backups
- Project bundle exports and planner backups
- Restore compatibility summaries
- Multi-language support (English, German, Spanish, French, Italian)
